# 🎧 Spotify Data Analysis Project

## 📌 Overview
This project performs exploratory data analysis on Spotify dataset to understand factors affecting song popularity.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Key Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis
- Data Visualization
- Popularity Prediction Model

## 💡 Insights
- Energy and danceability influence popularity
- Certain artists dominate high popularity tracks
